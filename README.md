Student Management System
Project Description
This project is a simple Student Management System created using Java programming language.  
It demonstrates Object-Oriented Programming (OOP) principles and basic data handling using ArrayList.

Purpose of the Project
The main goal of this project is to manage student information, store their grades, and calculate their GPA.  
It also allows sorting students based on their academic performance.

Features
 Add new students
 Store student grades using ArrayList
 Calculate GPA (average score)
 Sort students by GPA (from highest to lowest)
 Display student information
Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Sorting (Comparator / Lambda)

Project Structure
- Student.java → Student class (name, id, grades, GPA calculation)
- Main.java → Main program (adding, sorting, displaying students)

How It Works
1. Students are created in the program
2. Grades are added to each student
3. GPA is calculated automatically
4. Students are sorted by GPA
5. Results are displayed in console
